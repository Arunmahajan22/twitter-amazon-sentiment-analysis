# 🐦 Twitter & Amazon Sentiment Analysis

## 📌 Overview
This project implements **Sentiment Analysis using NLP** to classify text data (tweets and product reviews) into **positive or negative sentiment**.

It demonstrates a complete NLP pipeline including preprocessing, feature extraction, model training, and evaluation.

---

## 🎯 Objectives
- Analyze user opinions from text data
- Classify sentiment as Positive or Negative
- Compare performance across datasets

---

## 📂 Datasets Used
- Twitter Dataset (tweets)
- Amazon Reviews Dataset

---

## ⚙️ Techniques Used
- Text Preprocessing (punctuation removal, stopwords removal)
- Tokenization
- Count Vectorization (Bag of Words)
- Naive Bayes Classifier
- Logistic Regression (for comparison)

---

## 📊 Workflow
1. Data Loading  
2. Exploratory Data Analysis (EDA)  
3. Text Cleaning  
4. Feature Extraction  
5. Model Training  
6. Evaluation (Accuracy, Confusion Matrix, Classification Report)

---

## 🧠 Model
- **Naive Bayes** → Fast and efficient for text classification  
- **Logistic Regression** → Used for comparison  

---

## 📈 Results
- Achieved good accuracy on both datasets  
- Preprocessing significantly improved performance  

---

## 🚀 How to Run
1. Open the notebook in Google Colab  
2. Upload datasets (or use provided links)  
3. Run all cells step by step  

---

## 📌 Conclusion
This project shows how raw text can be transformed into meaningful insights using NLP and machine learning.
